# Cardiovascular Disease Detection System

## Description

The Cardiovascular Disease Detection System is a web-based healthcare application developed using Django to assist in the early assessment of cardiovascular disease risk.  

The system analyzes patient information such as age, gender, blood pressure, cholesterol, glucose level, body measurements, and lifestyle factors to estimate the risk of cardiovascular disease.

It provides a user-friendly interface that allows users to register, log in, view prediction history, and manage appointments. The system aims to support preventive healthcare through early risk detection and efficient patient record management.

---

## Features

- User registration and authentication
- Cardiovascular disease risk prediction using health indicators
- Patient dashboard with prediction history and statistics
- Appointment management system
- Secure storage of patient records
- Profile management for users
- Responsive and user-friendly interface

##  How to Run the Project
```bash
1. Clone the repository:
git clone https://github.com/mariam601/cardiovasc-PROJECT.git
2. Navigate to the project folder:
cd cardiovasc-PROJECT
3. Install dependencies:
pip install -r requirements.txt
4. Apply migrations:
python manage.py migrate
5. Run the development server:
python manage.py runserver
6. Open the website in your browser:
http://127.0.0.1:8000/
```
